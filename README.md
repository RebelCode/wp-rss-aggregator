## Requirements

**Recommended**
* PHP 7+
* PHP `curl` extension
* PHP `simplexml` extension
* PHP `mbstring` extension
* PHP `gd` extension
* WordPress 5.0+

**Minimum**
* PHP 5.4+
* PHP `curl` extension
* PHP `simplexml` extension
* WordPress 4.8+
